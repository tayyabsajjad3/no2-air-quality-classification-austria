# DMP Tool Contributors Note

Date: 2026-05-30

This note records the clarification from the course team about the TU Wien DMP Tool access bug.

## Course-Team Clarification

The course team explained that it is not necessary to give all contributors direct access to the DMP in the DMP Tool. It is enough that one group member has access to the DMP record. The other group members only need to be listed as contributors in the DMP metadata.

## How We Should Apply This

For T4.1, T4.2, T4.3, and T4.4, one group member can complete the DMP Tool record and exports. The DMP should still list all four group members as contributors/creators where the form asks for them. This satisfies the contributor requirement without trying to work around the DMP Tool access bug.
