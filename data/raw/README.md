# Raw Data

This folder contains the raw nitrogen dioxide (NO2) monitoring data downloaded from the European Environment Agency (EEA) data hub.

The raw Parquet files are kept under their original EEA filenames to preserve source provenance. These filenames identify the source-provided station-specific files and should not be changed manually. Processed or derived datasets created from these files must follow the repository naming convention, for example `processed_no2_measurements.csv`.

## Source

- Publisher: European Environment Agency (EEA)
- Data hub: https://www.eea.europa.eu/en/datahub/datahubitem-view/778ef9f5-6293-4846-badd-56a29c70880d
- Access date: 18 March 2026
- Format: Parquet
- Number of raw files: 16
